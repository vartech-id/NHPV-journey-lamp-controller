Frontend (Vue 3 + Vite + Bun)

Overview
- Single-page app for the event flow (register, info pages, controls).

Setup
1) From `frontend/`:
   - `bun install`

Run
- `bun run dev`
- Default dev URL: `http://localhost:5173`

Build
- `bun run build`
- `bun run preview`

Notes
- Backend URL is set in `frontend/.env` (`VITE_API_URL`).
